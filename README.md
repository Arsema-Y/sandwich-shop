# BIG-BACKS Sandwich Shop

- This project is for a java course capstone 
- it is the point of sales application for BIG-BACKS, a custom sandwich shop.
- The  business is growing, and we need a way to automate the order
process
- This is my interpretation of the business's online customer-service.
